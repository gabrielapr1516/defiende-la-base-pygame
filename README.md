# 🚀 Defiende la Base - Modo Oleadas

¡Bienvenido a **Defiende la Base**! Un videojuego arcade de naves desarrollado en Python utilizando la librería **Pygame**. El objetivo principal es proteger la Nave Nodriza de las incesantes hordas de enemigos espaciales a lo largo de varias oleadas y sobrevivir al temible jefe final.

---

## 🎮 Mecánicas del Juego

* **Sistema de Oleadas:** Enfréntate a oleadas de enemigos de dificultad progresiva antes de encarar el combate contra el Jefe Final.
* **Reparación de la Base:** Recolecta las 10 piezas de chatarra espacial que caen de los enemigos para desbloquear de forma permanente el sistema de defensas automáticas de la Nave Nodriza.
* **Interfaz HUD:** Controla tus vidas restantes y la salud de la base en tiempo real desde la pantalla.

## 🕹️ Controles

* **Movimiento:** Flechas del teclado (Izquierda / Derecha / Arriba / Abajo).
* **Disparo:** `Espacio`.
* **Menú Inicial:** `Espacio` para comenzar la batalla.
* **Pantalla Final:** `R` para reiniciar la partida y buscar la revancha | `ESC` para salir.

---

## 🛠️ Cómo Jugar

1. Ve a la sección de **Releases** en este repositorio (a la derecha).
2. Descarga el archivo `Defiende_la_Base_v1.0_Windows.zip`.
3. Descomprime el archivo en cualquier carpeta de tu computadora.
4. Abre la carpeta y ejecuta el archivo **`main.exe`** para empezar a jugar. ¡No necesitas instalar nada más!
